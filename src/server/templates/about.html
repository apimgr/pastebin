What is {{.SiteTitle}}?

{{.SiteTitle}} is a fast, public paste sharing service. Share code snippets, configuration files, logs, and any text content instantly — no account required.

{{t .Lang "about.features"}}

  • Syntax highlighting for 20+ languages
  • Configurable expiry (1 hour to 2 years, or never)
  • Burn-after-N-views for ephemeral content
  • Public and unlisted paste visibility
  • Delete token — pastes are yours to remove
  • Full REST API with JSON support
  • Compatible with pastebin.com, microbin, and lenpaste CLIs
  • No accounts, no tracking, no ads

API Compatibility

This service is a drop-in replacement for pastebin.com, microbin, and lenpaste. Existing scripts targeting those services work without changes by pointing them at this server.
